================================================================================
  DEPLOYMENT OPTIONS
================================================================================

OPTION 1: LOCAL DEPLOYMENT
  Location: Your computer running npm
  Command: npm run production (in admin-panel folder)
  Access: http://localhost:3000
  Best for: Internal team, testing

OPTION 2: NETLIFY DEPLOYMENT (RECOMMENDED)
  Steps:
  1. Build: npm run build
  2. Deploy build folder to Netlify
  3. Configure custom domain
  4. Enable HTTPS
  5. Access via your domain

OPTION 3: FIREBASE HOSTING
  Steps:
  1. npm run build
  2. firebase login
  3. firebase init hosting
  4. firebase deploy
  5. Access via Firebase domain

OPTION 4: SERVER DEPLOYMENT
  Location: AWS, DigitalOcean, Heroku, etc.
  Method: Node.js with Express (server.js included)
  Command: npm run serve

================================================================================
  EMERGENCY PROCEDURES
================================================================================

IF LOGIN FAILS:
  1. Check Firebase Console for admin account
  2. Verify email and password
  3. Clear browser cache
  4. Restart npm server

IF JOBS DON'T APPEAR:
  1. Check tickets exist in Firebase
  2. Verify dates are current
  3. Check technician assignment

IF PAYMENTS DON'T SHOW:
  1. Verify payment records exist
  2. Check status field is "completed"
  3. Verify amounts are set

IF APP WON'T LOAD:
  1. Check npm server is running
  2. Check console for errors (F12)
  3. Verify Firebase connection
  4. Clear cache and refresh
