Q: Do I need Android Studio or can I use command line?
A: You need either Android Studio (recommended, easier) OR manually set up 
   JDK, Android SDK, and Gradle. Android Studio is simpler.

Q: Can I build on Mac or Linux?
A: Yes! The same commands work, just use different package managers for 
   prerequisites (brew on Mac, apt on Linux).

Q: How large is the APK?
A: Debug APK: ~10-15 MB. Release APK: ~8-12 MB (smaller, optimized).

Q: Will it work offline?
A: Mostly yes. App functions offline but won't sync to Firebase until online.

Q: Can I update the app code?
A: Yes! After modifying React code:
   1. npm run build
   2. npx cap sync android
   3. gradlew.bat assembleDebug
   Or just: QUICK_APK_BUILD.bat

Q: How do I build a Release APK for distribution?
A: In Android Studio: Build â†’ Generate Signed Bundle/APK
   Or command line: gradlew.bat bundleRelease

Q: What Android version does it support?
A: Android 6.0 and higher (API 23+). Target: Android 14 (API 34).

Q: How do I update the APK name or icon?
A: See APK_BUILD_GUIDE.md section on Android configuration.
